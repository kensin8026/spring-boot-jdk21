# Product Overview

Spring Boot JDK21 demo application showcasing modern Spring ecosystem features including:

- Event-driven architecture with Kafka and Kafka Streams
- Modular monolith design using Spring Modulith
- Reactive HTTP clients with WebClient
- Circuit breaker patterns with Resilience4J
- JDBC-based data persistence
- Production-ready monitoring via Spring Boot Actuator

This is a demonstration project exploring Spring Boot 4.x capabilities with Java 21.
